# CommitEd - Guides & Tutoriels

## 1. Gestion de Projet (Process)

- **`/scrum`** : Organisation des Sprints, Dailies, et la "Definition of Done".
- **`/github`** : Workflow des Pull Requests, Code Review, et gestion des Issues.

## 2. Architecture & Standards (The "Glue")

- **`/api-standards`** : Structure JSON, Codes HTTP, et contrat d'interface (Swagger).
- **`/database`** : Stratégie de migration SQL et données de test (Seeding).
- **`/security`** : Gestion des secrets (.env), Auth (JWT), et règles .gitignore.

## 3. Outils & Versioning (DevOps)

- **`/git`** : Commandes, résolution de conflits, et stratégie de branches.
- **`/docker`** : Docker Compose, gestion des réseaux entre micro-services.
- **`/typst`** : Cheat sheet pour les rapports.

## 4. Langages & Stacks (Implementation)

- **`/backend`** : Standards spécifiques pour Spring, Symfony et .NET.
- **`/frontend`** : Structure des dossiers React, Hooks et gestion d'état.
- **`/setup`** : Installation de l'environnement (Oracle XE, JDK, PHP, Node).
